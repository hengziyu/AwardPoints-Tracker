# 学生奖项评分系统 (命令行版)

这是一个简单的Java命令行应用程序，用于记录和评分学生的奖项。

## 项目简介

这个小工具可以帮助老师或管理员快速地为学生的奖项进行分类和打分。它从一个Excel文件读取学生和奖项的列表，然后通过一个交互式的命令行界面来完成评分工作。所有的进度都会被保存下来，并且可以导出和导入，方便在不同电脑上使用。

## 主要功能

- **数据加载**:
    - 从指定的Excel文件加载学生和奖项数据。
    - 支持从上次的进度继续工作。
    - 可以生成随机的测试数据。
- **交互式评分**:
    - 通过命令行菜单为学生逐个奖项评分。
    - 评分时会显示奖项名称和图片链接。
    - 用户只需输入数字选项即可完成评分。
- **进度管理**:
    - 所有评分进度都会自动保存到`Student_Awards.xlsx`和一个SQLite数据库(`student.db`)中。
    - 支持将所有进度**导出**为一个JSON快照文件。
    - 支持从快照文件**导入**进度，完全恢复当时的状态。

## 如何运行 (超简单指南)

本项目已经包含了运行所需的所有环境，你不需要在电脑上安装任何东西！

### 对于开发者

如果你修改了代码，需要重新编译项目。

1.  **准备环境 (仅需一次)**:
    *   确保项目根目录下有一个 `jdk` 文件夹，其中包含完整的 JDK 17 环境。
    *   确保电脑上安装了 Maven，并已配置好国内镜像（可以参考之前的指南）。

2.  **编译项目**:
    *   在项目根目录下打开命令行，运行：
        ```shell
        mvn clean package
        ```
    *   这会在 `target` 文件夹下生成最新的 `untitled-1.0-SNAPSHOT-jar-with-dependencies.jar` 文件。

### 对于普通用户

拿到这个项目文件夹后，直接运行即可。

1.  **Windows 用户**:
    *   直接双击项目根目录下的 `run.bat` 文件。

2.  **macOS / Linux 用户**:
    *   首先，需要给脚本添加执行权限。在项目根目录下打开终端，运行：
        ```shell
        chmod +x run.sh
        ```
    *   然后，运行脚本启动程序：
        ```shell
        ./run.sh
        ```

## 使用说明

程序启动后，你会看到一个彩色的命令行界面。

1.  **启动菜单**:
    - 首先，你需要选择数据源。对于第一次使用，建议选择 `1. 从新文件加载` 并提供你的原始学生数据Excel表。
    - 如果你之前已经运行过并生成了进度，可以选择 `2. 从上次文件加载`。

2.  **主菜单**:
    - **1. 列出所有学生及其进度**: 查看所有学生的列表和他们的评分进度。
    - **2. 为指定学生评分**: 输入学生的学号，进入评分模式。
    - **3. 导出进度**: 将当前所有学生的评分进度保存到一个JSON文件中。
    - **4. 导入进度**: 从一个之前导出的JSON文件恢复所有进度（注意：这会覆盖当前的所有数据！）。
    - **5. 退出**: 退出程序。

3.  **评分界面**:
    - 进入评分模式后，程序会逐个显示该学生还未评分的奖项。
    - 你会看到奖项的名称和图片链接。
    - 根据提示，输入对应的数字来为奖项选择一个分类（如国奖、省奖等）。
    - 你也可以直接按回车跳过当前奖项，或者输入 `q` 退出评分。

## 清理项目 (恢复如初)

如果你想将项目文件夹恢复到最原始、最干净的状态，可以使用我们为你准备的清理脚本。

这个脚本会删除所有在编译和运行过程中产生的文件，包括：
- `target` 编译目录
- `app.log` 日志文件
- 所有自动生成的 `.xlsx` 和 `.db` 文件
- 所有导出的 `snapshot` 快照文件

### 如何使用

- **Windows 用户**: 直接双击 `cleanup.bat` 文件。
- **macOS / Linux 用户**: 在终端中运行 `./cleanup.sh` (如果提示权限不足，请先运行 `chmod +x cleanup.sh` )。
